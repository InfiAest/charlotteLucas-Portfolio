Live Rock

19.04.2021 - 23.05.2021

This rock blog site was the project exam at the end of the first year studying at Noroff. We had to use everything we had learnt through the year to design and build a blog site with a theme of choice.

Brief

To create a blog site, the topic our own choice, and it must include a carousel(slider) that shows the latest posts, a button which would load the next 10 posts available, modals which display images full-sized, a well validated contact form and the data should be stored on a WordPress API so the data can be fetched from there.

Extra tasks to complete were adding a sort, filter or search function to the blogs page which I completed and allowing users to submit comments and posting that data to WordPress which I have yet to complete.

Tools used -

Adobe XD, Adobe Photoshop, HTML, CSS, Vanilla JavaScript, WordPress API, GET requests.

My thoughts

I am happy with the aesthetics of the finished website. I feel it really fits the topic of the site and looks great on all screen sizes. I am yet to add the ability to comment on posts but I am happy with the way everything else works on the site.
