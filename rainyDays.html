Rainy Days

24.08.2020 - 21.03.2021

This is a cross-course project assigned at Noroff. We were given this as our first big project, where we started with the design, built it using HTML & CSS, then went back to it after learning JavaScript and Interaction Design and added finishing touches after trying to read through the mess of first timers code.

Brief

Target Audience - Men and women aged 30 to 50 whose interests are: being outdoors, hiking, exploring, skiing, camping, canoeing

Build an ecommerce style site for the chosen client. The site architecture should be; home, list of all jackets, a jacket specific page, a checkout page, a checkout success, about page and contact page.

Create a WordPress API where all the product content can be stored and call to it in order to reduce loading times and reduce the amount of hardcoded HTML as well as making it function like a real-world e-commerce site.

Tools used -

Adobe XD, Adobe Photoshop, HTML, CSS, Vanilla JavaScript, Wordpress API, GET Requests.

My thoughts

I'm happy with the design and responsiveness of the finished website. Hovever, the code is a work in progress which I am continuing to fix and make better. It's the first website I ever made and it holds a special place in my heart.
