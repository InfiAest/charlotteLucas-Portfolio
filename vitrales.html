Vitrales Interior

18.10.2021 - 14.11.2021

This project was only a design project in the second year at Noroff where we were asked to design a homepage and product page for an Interior Design site where we took our own product photographs and edited them in photoshop to suit the theme of the website.

Brief

For an interior design site design the logo, choose appropriate typography, layout, colours and create a brand personality and target audience. Take photographs and edit them to use in the site. Practice using CSS filters as well as Photoshop to achieve the desired look.

Tools used -

Adobe XD, Adobe Photoshop, HTML, CSS.

My thoughts

I liked the design so much that I ended up building the two pages into a live site with basic HTML and CSS and will continue to add to it in my spare time.
