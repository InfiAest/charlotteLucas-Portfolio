Bark Life

15.11.21 - 19.12.21

Bark Life was a semester project in my second year at Noroff. This website was to challenge us to use any css frameworks and new JavaScript that we had learnt during the first semester.

Brief

We were tasked to create an e-commerce website that has both customer facing and admin sections using a Strapi API. We could choose the theme of the website, so I chose a dog boutique because well, dogs!

Checkout and payment systems were not necessary in this assignment, but we had to include admin pages where we could add, delete and update products. We also had to include a functioning product cart and search/filter function.

Tools used -

Adobe XD, Adobe Photoshop, HTML, SCSS, Strapi API, GET/PUSH/PUT/DELETE Requests, Local storage, Heroku and Cloudinary.

My thoughts

I'm very happy with the outcome of the Bark Life website and my code for it. The biggest challenge I overcame was connecting the Strapi API to Heroku and Cloudinary allowing for the product images to be saved to the database and then updating the product images using a PUT request. It was a super fun project to work on and turned out to be really cute!
